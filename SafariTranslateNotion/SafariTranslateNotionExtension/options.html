DeepSeek API

Get your key from DeepSeek Platform.

Notion

Go to Notion Integrations → create an internal integration → copy the API secret (starts with ntn_) from the Configuration tab. Then open your database page in Notion, click ⋯ → Add connections → select your integration.

Open the database as a full page in Notion. The ID is the 32-character hex string in the URL: notion.so/{ID}?v=…

Obsidian Vault

Pick your Obsidian vault folder to grant the extension permission to write notes.

Notes will be written to <vault>/<folder>/<word>.md (default: vocab).
